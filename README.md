# arrowtrack-legal

Public legal documents for the **ArrowTrack** mobile app. Published via
GitHub Pages at <https://mjacobsen71.github.io/arrowtrack-legal/>.

## Contents

- [`privacy-no.md`](./privacy-no.md) — Norsk personvernerklæring
- [`privacy-en.md`](./privacy-en.md) — English Privacy Policy
- [`index.md`](./index.md) — Landing page with language selection

## Why a separate repo?

The main ArrowTrack source repository is private. GitHub Pages on the free
tier requires a public repo. Privacy Policy must be publicly reachable for
Apple App Review and end users. Hosting only the legal documents here keeps
the source code private while satisfying both requirements.

## Updates

The canonical source for these documents is
`docs/legal/` in the private main repo. Push updates here when the policy
changes — typically when new third parties are introduced (Sentry, RevenueCat,
etc.) or when monetization launches.
